# FutbolChimera

**FutbolChimera** is the mobile home of Chimera for iOS and Android. It was extracted from [`localchimera/localchimera`](https://github.com/LocalChimera/localchimera) and now lives at [`github.com/terexitariusstomp/futbolchimera`](https://github.com/terexitariusstomp/futbolchimera).

The project ships two mobile app codebases and the soccer-focused AI, analytics, and P2P tooling that the mobile app surfaces.

## Apps

### `apps/mobile-expo` (current)

A React Native + Expo mobile app that wraps a shared React frontend in a WebView. The native side exposes local file-system, machine-id, and deep-link handling, plus a set of `/api` endpoints that the frontend can call.

Key mobile features:

- **iOS & Android builds** — native projects generated by Expo prebuild
- **Adaptive icons & splash screens** — configured in `app.json`
- **On-device AI** — originally QVAC/BareKit; currently uses `@localchimera/sdk` stack
- **Deep-link wallet callbacks** — for Privy/WalletConnect auth flow
- **P2P swarming** — `/api/swarm/*` endpoints for status, create, and join
- **Soccer AI coach** — `/api/soccer/ai-coach` and `/api/soccer/ai-chat`
- **Analytics reference** — `/api/soccer/reference`, `/api/soccer/visualize`
- **Computer vision** — `/api/soccer/detect`, `/api/soccer/models/*` for player/ball detection

See [`apps/mobile-expo/README.md`](apps/mobile-expo/README.md) for build instructions.

### `apps/mobile` (legacy)

A Capacitor-wrapped mobile app. Each phone was intended to run as a standalone Chimera node. This directory is kept for reference and historical continuity.

## External resources

The repo includes submodules that provide soccer analytics and AI coaching reference material:

- `external/ai-soccercoach` — AI soccer coaching assistant
- `external/analytics-handbook` — soccer analytics concepts and code
- `external/football-analytics-tutorials` — football analytics tutorials
- `external/roboflow-sports` — computer vision models for sports

Run `git submodule update --init --recursive` after cloning to pull them in.

## Templates

The `templates/` folder contains markdown templates used by the AI coach and analytics features in the mobile app:

- `match-analysis.md` — post-match review structure
- `player-evaluation.md` — player scouting and development
- `tactical-plan.md` — pre-match tactical planning

## Quick start

```bash
# Clone with submodules
git clone --recursive https://github.com/terexitariusstomp/futbolchimera.git
cd futbolchimera

# Install and run the Expo app
cd apps/mobile-expo
npm install
npx expo start
```

## Build

```bash
cd apps/mobile-expo

# Android
npx expo prebuild --platform android --clean
cd android
./gradlew assembleRelease

# iOS
npx expo prebuild --platform ios --clean
npx expo run:ios
```
