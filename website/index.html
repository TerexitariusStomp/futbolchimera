Built for the pitch

Every feature is designed to give players, coaches, and analysts an edge.

⚽

AI Coach

Ask tactical questions, get minute-by-minute analysis, and receive substitution recommendations powered by local soccer prompts.

📊

Analytics Reference

xG, passing networks, pitch control, heat maps, and shot maps — all grounded in the analytics-handbook concepts.

Computer Vision

On-device player and ball detection, jersey-number OCR, and pitch keypoints via Roboflow Sports models.

Create or join a Pear/Hyperswarm network to sync wiki pages, match notes, and tactics directly between phones.

Cross-platform

On-device AI

Build it yourself

Clone the repo, install dependencies, and run the Expo app locally.

terminal 
git clone --recursive https://github.com/terexitariusstomp/futbolchimera.git
cd futbolchimera/apps/mobile-expo
npm install
npx expo start

# Android
npx expo prebuild --platform android --clean
cd android && ./gradlew assembleRelease

# iOS
npx expo prebuild --platform ios --clean
npx expo run:ios
